# PROMPT: Cytology Analysis with User-Friendly Explanations

## INPUT DATA:
- Image: [provided as binary inline_data]
- Per-cell features (JSON): $FEATURES_JSON
- Predictions (per cell): $PREDICTIONS_JSON
- Prediction probabilities (per cell; order [HSIL, LSIL, NSIL]): $PROBS_JSON
- Slide-level final class (DO NOT MODIFY): $OVERALL_CLASS
- Slide-level probabilities (order [HSIL, LSIL, NSIL]): $OVERALL_PROBS_JSON

## CLASS MAPPING:
- 0 → HSIL (High-grade Squamous Intraepithelial Lesion)
- 1 → LSIL (Low-grade Squamous Intraepithelial Lesion)
- 2 → NSIL (Negative for Squamous Intraepithelial Lesion)

## TASK INSTRUCTIONS:

### 1) PER-CELL ANALYSIS:

For each cell in $PREDICTIONS_JSON, generate:

**a) Classification:**
   - State predicted class (HSIL/LSIL/NSIL)

**b) User-friendly explanation (40-60 words):**
   
   **Structure your explanation as follows:**
   
   - **Opening statement:** Begin with accessible language
     - Example: "This cell shows characteristics of [class]..."
     - Example: "Analysis indicates this is a [class] cell..."
   
   - **Key findings:** Explain the THREE primary features in plain language:
     - **NCr (Nucleus-to-Cytoplasm ratio):** 
       * "The nucleus occupies [percentage/description] of the cell"
       * "Shows [enlarged/normal/small] nucleus relative to cytoplasm"
     
     - **Solidity (Nuclear membrane regularity):**
       * "The nuclear border appears [smooth/irregular/highly irregular]"
       * "Nuclear contour shows [minimal/moderate/significant] irregularity"
     
     - **N (Nuclear area):**
       * "Nuclear size is [within normal limits/enlarged/markedly enlarged]"
       * "The nucleus measures [description relative to normal]"
   
   - **Supporting details (alwaya add some other features):**
     * Add context from other features when they strengthen the diagnosis
     * Use comparative language: "consistent with", "supporting", "additionally shows"
   
   - **Clinical context:** Relate findings to diagnostic significance
     - Example: "These features are typical of [class] cells"
     - Example: "This pattern suggests [clinical interpretation]"

   **Tone guidelines:**
   - Use clear, professional language without excessive jargon
   - When technical terms are necessary, briefly explain them
   - Avoid absolute statements; use "indicates", "suggests", "consistent with"
   - Focus on observable facts rather than speculation

**c) Confidence assessment:**
   - Calculate as: max($PROBS_JSON[cell_id])
   - Express as percentage in explanation when relevant

---

### 2) SLIDE-LEVEL SUMMARY:

Generate a comprehensive, accessible summary:

**a) Overall classification:**
   - Echo $OVERALL_CLASS exactly (no modifications)

**b) Narrative explanation (60-100 words):**
   
   **Structure as follows:**
   
   - **Opening statement:**
     * "This slide is classified as [overall_class]..."
     * Immediately state the conclusion clearly
   
   - **Cell distribution:**
     * "Of the [total] cells analyzed, [X] were classified as [class], [Y] as [class]..."
     * Use actual numbers for transparency
   
   - **Confidence and evidence:**
     * Reference $OVERALL_PROBS_JSON to explain certainty level
     * "The classification shows [high/moderate/low] confidence ([percentage]%)"
     * Highlight most diagnostically significant cells
   
   - **Key diagnostic findings:**
     * Mention the strongest evidence supporting the diagnosis
     * "The diagnosis is based primarily on [specific findings]"
     * If relevant: "Notable findings include [X high-confidence HSIL cells with NCr >0.80]"
   
   - **Concordance notes (if applicable):**
     * Address any apparent discrepancies transparently
     * Example: "While most cells appear normal, the presence of [X] high-grade cells determines the overall classification"
     * Example: "The diagnosis reflects [clinical reasoning] despite [observation]"
   
   **Tone for slide summary:**
   - Write as if explaining to a healthcare provider who needs actionable information
   - Balance technical accuracy with accessibility
   - Be transparent about uncertainty or complexity
   - Provide clinical context for decision-making

**c) Aggregate confidence (from $OVERALL_PROBS_JSON):**
   - High: max_prob ≥ 0.70
   - Moderate: 0.50-0.69
   - Low: < 0.50

**d) Diagnostic notes:**
   - Flag critical observations
   - Note any limitations or considerations
   - Highlight cells requiring special attention
